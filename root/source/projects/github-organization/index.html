Over time I seem to have gathered lots of Github repositories. Enough that I found my user page becoming cluttered and hard to manage. In an attempt to make things less messy, I have created several Github organizations. I use the "organizations" as folders to organize my repositories.

neil-smithline: My user page. I am actively working on these repos. They may be in an experimental, non-working state.

neil-smithline-elisp: Original Emacs lisp projects that are in a relatively stable state and available for use by others.

neil-smithline-shellscripts: Original shell scripts that are in a relatively stable state and available for use by others.

neil-smithline-older-projects: Original projects that I am no longer working on.

neil-smithline-published: Original projects that I try keep more stable than repos in other repositories.

neil-smithline-just-watching: Repositories that I have forked but am not really working on. These are repositories written by other people.
